Jaký je cíl této stránky?

Náš web je zaměřen na vysvětlení konceptu Machine Learningu, s důrazem na LLM (Large Language Models). Poskytujeme materiály pro studenty, které jim pomohou pochopit základy a aplikace těchto technologií.

Web nabízí interaktivní procvičování a studijní materiály pro žáky a pro učitele je připravený soubor nástrojů pro efektivní výuku.

Druhy Machine Learningu

Supervised Learning: Model je trénován na označených datech, kde každá vstupní data mají odpovídající výstup. Cílem je naučit model předpovídat výstup na základě nových vstupních dat.

Unsupervised Learning: Model pracuje s neoznačenými daty a snaží se najít skryté vzory nebo struktury, například pomocí shlukování nebo redukce dimenzionality.

Reinforcement Learning: Model se učí prostřednictvím interakce s prostředím. Za správné akce dostává odměny a za špatné akce penalizace, což ho motivuje k lepším výkonům.
