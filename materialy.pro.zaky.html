Úvod do Machine Learningu

Druhy Machine Learningu

Supervised Learning

Model je trénován na označených datech, kde každá vstupní data mají odpovídající výstup. Cílem je naučit model předpovídat výstup na základě nových vstupních dat.

Příklad: Rozpoznávání rukopisných písmen, kde jsou trénovací data obsahující písmena a jejich správné označení.

Unsupervised Learning

Model pracuje s neoznačenými daty a snaží se najít skryté vzory nebo struktury, například pomocí shlukování nebo redukce dimenzionality.

Příklad: Shlukování zákazníků podle jejich nákupního chování.

Reinforcement Learning

Model se učí prostřednictvím interakce s prostředím. Za správné akce dostává odměny a za špatné akce penalizace, což ho motivuje k lepším výkonům.

Příklad: Trénování robota k navigaci v prostoru pomocí pokusů a omylů.

Classification in Supervised Learning

Classification (klasifikace) je jedním z nejběžnějších typů supervised learning (učené s učitelem) v oblasti strojového učení. Tento typ strojového učení se používá k přiřazení vstupních dat do jedné nebo více kategorií (nebo tříd) na základě historických dat, která byla označena (tzn. obsahují správné odpovědi).

Jak funguje Classification? Představ si situaci, kdy máme dataset s obrázky koček a psů. Tento dataset obsahuje obrázky (vstupní data) a jejich odpovídající označení, tj. každému obrázku je přiřazeno, zda jde o kočku, nebo psa. Cílem klasifikačního algoritmu je naučit se rozlišovat mezi těmito dvěma kategoriemi.

Proces Klasifikace:

  • Trénování: Model se trénuje na označených datech, aby našel vzory pro přiřazení nových vstupních dat do kategorií.
  • Testování: Model je testován na neoznačených datech, aby ověřil svou schopnost správně klasifikovat nové informace.
  • Predikce: Model predikuje kategorie pro nové vstupy na základě vzorů naučených při trénování.

Příklady:

  • Detekce e-mailů jako spam nebo ne-spam.
  • Diagnostika nemocí z lékařských snímků.

Regression in Supervised Learning

Regression (regrese) je další populární technikou v supervised learning (učené s učitelem), která se používá k předpovědi číselných hodnot. Na rozdíl od klasifikace, kde model přiřazuje vstupy do kategorií, regresní modely předpovídají hodnoty, které spadají do nějakého rozsahu čísel.

Jak funguje Regression? Představte si, že máme dataset, který obsahuje údaje o velikosti domu (v m²) a jeho ceně. Cílem regresního modelu je najít vztah mezi velikostí domu a jeho cenou, aby mohl na základě těchto informací předpovědět cenu nového domu.

Proces Regrese:

  • Trénování: Model se trénuje na historických datech, kde jsou známé vstupy (např. velikost domu) a odpovídající výstupy (např. cena). Model se naučí vztah mezi těmito hodnotami.
  • Testování: Model je testován na nových datech, aby ověřil, jak dobře dokáže předpovědět cenu na základě nových vstupů.
  • Predikce: Model predikuje hodnotu pro nové vstupy, například cenu nového domu na základě jeho velikosti.

Příklady:

  • Předpověď cen nemovitostí na základě jejich velikosti a dalších parametrů.
  • Předpověď prodeje produktu na základě historických dat.
  • Předpověď teploty na základě klimatických dat.

Clustering in Unsupervised Learning

Clustering (shlukování) je technika strojového učení v rámci unsupervised learning (učení bez učitele), která se používá k seskupování podobných dat do shluků (nebo skupin). Na rozdíl od klasifikace, kde máme označená data, clustering nevyžaduje žádná předchozí označení. Model se učí pouze na základě podobnosti mezi daty.

Jak funguje Clustering? Představ si situaci, kdy máme dataset se zákazníky a jejich nákupním chováním. Bez předchozích informací o tom, kdo je kdo, model seskupí zákazníky podle podobnosti jejich nákupů do několika shluků.

Proces Shlukování:

  • Shlukování: Model se pokusí najít skupiny podobných datových bodů, aniž by měl k dispozici informace o jejich identitě.
  • Analýza: Po vytvoření shluků je možné analyzovat, jaké charakteristiky mají jednotlivé skupiny.

Příklady:

  • Segmentace zákazníků podle jejich nákupního chování.
  • Shlukování dokumentů nebo textů podle témat.
  • Identifikace podobných obrazů v databázích.

Reinforcement Learning

Reinforcement learning (RL) je oblast strojového učení, která se zaměřuje na učení prostřednictvím interakce s prostředím. V RL se agent učí, jak se chovat v určitém prostředí, aby maximalizoval odměnu, kterou dostává za své akce. Tento přístup je podobný tomu, jak se lidé učí z pokusů a omylů.

Jak funguje Reinforcement Learning? Představ si agenta (např. robota), který se nachází v nějakém prostředí (např. labyrintu). Tento agent provádí různé akce a za správné akce dostává odměny, zatímco za nesprávné akce dostává penalizace.

Proces Reinforcement Learning:

  • Agent: Subjekt, který se učí a rozhoduje se na základě akcí.
  • Prostředí: Místo, kde agent interaguje, např. virtuální svět nebo robotické zařízení.
  • Odměny a Penalizace: Odměny motivují agenta k užitečným akcím, penalizace ho odrazují od špatných rozhodnutí.

Příklady:

  • Trénování robota, který se učí procházet překážkami v prostředí.
  • Hraní her (např. šachy nebo Go), kde agent zlepšuje své schopnosti prostřednictvím herní strategie.
  • Automatizované řízení ve výrobních linkách, kde se agent učí optimalizovat procesy.

Úvod do LLM

Co je to LLM?

LLM (Large Language Model) je typ neuronové sítě, která je trénována na obrovském množství textových dat, aby dokázala generovat lidsky srozumitelný text, odpovídat na otázky nebo vytvářet souvislé příběhy.

Příklad: Chatboti jako ChatGPT, kteří umí odpovídat na složité dotazy.

Jak se trénuje LLM?

Model se učí na základě textových dat a pomocí metody zvané transformer. Během trénování se snaží předpovědět další slovo v textu a postupně se zlepšuje díky zpětné vazbě.

Příklad: Model čte větu "Dnes je krásný ..." a snaží se předpovědět další slovo (např. "den").

Kde se LLM používají?

LLM se využívají v různých oblastech, jako jsou virtuální asistenti, překladače nebo nástroje pro automatické generování textu.

Příklad: Google Translate nebo hlasoví asistenti jako Siri.

Neuronová síť

Základem LLM je umělá neuronová síť, která napodobuje způsob, jakým funguje lidský mozek. Obsahuje vrstvy neuronů, které zpracovávají informace a učí se z dat.

Proces učení:

  • Vstupní vrstva: Přijímá textová data.
  • Skryté vrstvy: Analyzují a hledají vzory v datech.
  • Výstupní vrstva: Generuje odpověď na základě naučených vzorů.

Jak se LLM učí?

Trénování LLM probíhá ve třech hlavních fázích:

  • Předtrénování: Model se učí na obrovském množství dat.
  • Doladění: Upravuje se pro konkrétní úkoly (např. zákaznická podpora).
  • Interakce s uživateli: Model se postupně zlepšuje díky zpětné vazbě.

Příklad: Model se nejprve učí obecné informace a poté se specializuje na lékařské poradenství.

Rizika a etika LLM

Použití LLM přináší nejen výhody, ale i rizika, jako je šíření dezinformací nebo zkreslené odpovědi. Je důležité zajistit, aby byly modely používány eticky a bezpečně.

Příklad: Filtry obsahu, které zabraňují šíření škodlivých informací.
